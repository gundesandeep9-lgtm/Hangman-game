 Hangman Game - CodeAlpha Python Internship (Task 1)

 Project Overview

This project is developed as **Task 1** for the **CodeAlpha Python Programming Internship**.

The Hangman Game is a simple console-based Python application where the player has to guess a randomly selected word one letter at a time. The player is allowed a maximum of **6 incorrect guesses** before the game ends.

---

 Features

- Randomly selects a word from a predefined list.
- Displays hidden letters using underscores (_).
- Accepts one letter at a time from the user.
- Prevents duplicate guesses.
- Allows only 6 incorrect guesses.
- Displays a winning message when the word is guessed.
- Displays the correct word if the player loses.
